The **Business Model Canvas** is a strategic management tool used to visually develop and describe a business model. It includes key building blocks such as:

- **Key Partners**
- **Key Activities**
- **Value Propositions**
- **Customer Relationships**
- **Customer Segments**
- **Key Resources**
- **Channels**
- **Cost Structure**
- **Revenue Streams**
